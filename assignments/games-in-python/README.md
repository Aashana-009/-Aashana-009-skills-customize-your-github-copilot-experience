# 📘 Assignment: Games in Python

## 🎯 Objective

Build a classic Hangman game in Python using strings, loops, conditionals, and random selection. This assignment will help you practice core programming concepts while creating a fun text-based game.

## 📝 Tasks

### 🛠️ Create the Core Game

#### Description
Implement the main gameplay loop for a Hangman game using the starter code provided in this folder.

#### Requirements
Completed program should:

- Randomly choose a secret word from a predefined list.
- Show the current word progress using underscores for unrevealed letters.
- Accept one letter at a time from the player.
- Reveal correct letters and track incorrect guesses.
- End the game when the player wins or runs out of attempts.

### 🛠️ Improve the User Experience

#### Description
Add simple game feedback to make the experience clear and engaging for the player.

#### Requirements
Completed program should:

- Display the number of incorrect guesses remaining.
- Show the letters the player has already guessed.
- Print clear win and lose messages at the end of the game.
- Use readable code with helpful comments where needed.
